My Games

These Games are a mixture of Unity and Unreal Engine games which I have either fully created or been a part of the group which has created. My roles are usually Coding however sometimes it has been art or other aspects.

This Game was created in a group where we created a series of minigames and then switch our roles. For this game I created all the visual art and animation for the game in blender then ported it over into Unity. I also helped brainstorm the idea for the game and wrote the initial code for the pigs ai which I had written as a state machine. This game took us about 1 month to make and was the first game we did in that class.

This Game was created in the same group as the previous game except the goal for this game was to create a multiplayer game. For this game we used PUN2 for unity for the multiplayer servers and I wrote the code for getting the players able to see each other and connect to the lobby and then the main game. This game taught us however that when creating a multiplayer game like this to write code specificly for the game rather then take something we already had. This is because we only had 3 weeks for this game and were creating it during midterms for our other class so we used code we had from a previous game. However since that code wasn't written for multiplayer we wasted to much time trying to get it to work that we couldn't finish the game. However since the multiplayer aspects did work we consider it a success.

This was the first Unreal Engine game which I ever created and it is a basic target shooter. My group learned from the previous game and tried to keep this one simple so as to have a very polished game as possible. For this game I created the targets code and the score system.

This last game was the second game I worked on in Unreal Engine it was inspired by games like SAS zombie assault. I created all the assets in blender and used this game to learn the blueprint system as much as I could.
