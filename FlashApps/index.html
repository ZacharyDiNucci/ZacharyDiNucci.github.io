Flash Apps

I made these flash applications in my first two years of GIMM and they were made using Adobe Animate.

This Game was programmed in my GIMM 110 class and was suppost to be a simple shooter, it was inspired by classic games like 1942. It is a castle seige where you can lose however if you survive long enough you delay the enemy long enough to win. It was the first game which I created in the class and while I had problems at time it was really fun to create.

DungeonCrawl was created as a group project where I helped worked on the code for the enamies. This project started as a demo state machine game which we were given and told we had to make an actual game out of it. We were inspired by D&D and wanted to create a game where the hero had to make it through a dungeon. As of now it still has some bugs such as not fully removing the zombies when killed, however I hope to go back and improve everything in it.
