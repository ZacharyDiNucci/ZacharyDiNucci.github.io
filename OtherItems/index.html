Other Items

This page is going to hold a few different to projects I have worked on such as art and music.

This drawing was done for an assignment where we had to take a picture of an area of Boise and make a post apocolyptic scene. I chose a bridge which is close to my house because in the mornings the river would be covered in a very thick fog. I felt like this would fit the theme and a partially collaped bridge would be a fun project. This was created in Adobe Photoshop using a drawing tablet and took some time to create however I had alot of fun with this project and felt is is some of the best art I have done and helped me learn how to do some of the basics of photoshop.
