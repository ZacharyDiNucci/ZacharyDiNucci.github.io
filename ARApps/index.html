AR Creations

These demonstrations are from 2 different projects which involve helping teach Middle School students with autism how to better interact with the world. These projects were created with Unity and C# and were designed to work on both iPhone and Android.

This first project was the first AR project which I worked on and was to teach social distancing to students with autism. The original premise was how close to stand near someone depending on if you know them or not. There is provided feedback and points to better help the student know how well they are doing. The UI was temporary since it was just a pitch it is a minimum viable product with my part of the slice being this ar part with my groupmates working on other aspects like menu and sql stuff.

This project is less of a game and more to teach the steps for students with autism to take to ride a local bus. This project is still ongoing and hasn't yet gotten final art however I had to learn some of the quirks of unity like objects cannot be to large so I broke up the bus into many smaller objects to work. This project has been a class project with about 15 other students where we are making a VR game, an AR App, and 2D videos. We are working with some local teachers with this in learn how to work with a client for such projects
