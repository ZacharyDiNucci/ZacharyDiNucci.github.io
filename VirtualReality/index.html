Virtual Reality

An important part of the GIMM major is to create a VR game which we will have finished by the end of the degree. This game is currently under development and is about children in a food fight. To create this game I have used the Unity Engine and developed an alpha. Using an Oculus Rift as the headset I created basic "shooting", bots, ammo/health reloading, and other basic systems. Using Blender I am creating the main assets which I am working on importing into Unity.

In my 2nd year of GIMM I had to create a proposal for my game and in the 3rd year I began creating my game. In the Images above I show what I currently have.
