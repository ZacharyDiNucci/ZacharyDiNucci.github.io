Virtual Reality

An important part of the GIMM major is to create a VR game which we will have finished by the end of the degree. My Game is about children in a food fight and uses the Valve Index Controllers in order to use the hand shapes for throwing food. The art was designed to be simple and cartoon like. There is a basic class customization system by actually changing the classes you are in.
