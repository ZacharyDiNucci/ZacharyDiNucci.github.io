Multiplayer Bomber

UNITY MULTIPLAYER GAME/ Fall 2020

About this Game

Choose your side, are you the easy to maneuver but hard to aim air crafts or the hard to move but powerful AA cannon. In this multiplayer fight its 1 v 3 as we see how can win and if numbers can overcome or will power win.

About the Development

Multiplayer Bomber was the third game created with the group who created PIG and Quack Attack. This game was to be a Unity Multiplayer game using Photon to have 2 teams battle for domination. This game didn’t work as we hoped because we were using stuff from our second game which used machine learning to teach bots to attack the player. However, those assets were not built with Photon in mind and so their interactions inevitably failed. However, I still was able to make a multiplayer game with a lobby where the players would connect to each other using Photon. I think of this game as a lesson for how to create a game using Pun and I plan to make a new Multiplayer game soon.
