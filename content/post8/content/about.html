UNITY THIRD PERSON SHOOTER/Spring/Summer 2022

About this Game

This was a demo I created to test some of the newer Unity 3d Systems which had come out since I started developing in Unity. I created this Third Person Shooter to learn Cinamachines, Unity’s animation systems, and some Gun Logic which I hadn’t learned yet. While I don’t plan to come back to this project, I do want to use the animation knowledge in the current First Person Shooter I am working on for the reloading, movement system, and some other small things. The art in this project was from a Polygon pack I got to better learn this without having to worry about modeling my own art and if it would look good.
