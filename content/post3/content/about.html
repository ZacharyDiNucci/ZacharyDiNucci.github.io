UNITY TOP DOWN GAME/ Fall 2020

About this Game

Life in the country side is normally pretty easy. Today is the exception, the crops need planting but the local boar population has been exploding. Defend your corn from these pests and when it is done growing deposit it in the carts. Watch out though, the boars will defend themselves and some have sharp tusks.

About the Development

PIG was developed as a group project in one of my GIMM classes. I came up with the idea for the game and was tasked with doing all the art and visuals. For this I created the UI, 3d models, and textures. The models and textures were created in Blender and these were the first models I had ever rigged in Blender. For the animations on the Pigs, I used Blender’s animation tools and exported them to Unity. For the player I used Mixamo by Adobe to create the run and attack animation.

The pigs in this game run off a state machine for their AI. This was something I was also able to help with by providing a tutorial I found for Food Flingers on State Machines. I also provided some basic state machine starting code for the two pigs. This project went really well and I feel proud of my little piggies I made for the game.
